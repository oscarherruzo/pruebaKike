¡Hola! Soy Enrique Contreras López, desarrollador Backend especializado en Java y el ecosistema Spring. Empecé construyendo interfaces con HTML, CSS y Angular, pero rápidamente me di cuenta de que lo que más me apasiona está detrás del botón — la lógica, la arquitectura, los sistemas que no se ven pero lo sostienen todo.

Hoy trabajo como Backend Developer en Nter Tech Services, con BBVA como cliente, desarrollando estructuras Batch, Batch Scheduler y Daemons para el reporte de transacciones bancarias. Un entorno de alta exigencia donde cada proceso tiene impacto real en las operaciones del banco.

Llegué aquí pasando por tres etapas en Nter — prácticas formativas, contrato de formación y jornada completa — lo que me dio una base sólida: SQL, Git, Python, Spring Boot y ahora Spring Batch en producción. Antes de eso, tuve mi primer contacto profesional en GRUPO SOLUTIO con Angular y SCSS.
